# 高中数学三年知识点回放（含高频考点详解）

> 按"集合与逻辑 → 函数 → 代数 → 三角 → 向量 → 解析几何 → 立体几何 → 数列 → 不等式 → 概率统计 → 导数 → 选修"主线梳理。每节先列知识骨架，再用 **【高频考点】** 标注常考、易错、压轴点，附解题套路。

## 一、集合与常用逻辑用语

### 知识骨架
- **集合关系**：子集 $A\subseteq B$、真子集 $A\subsetneqq B$、相等。设有限集 $A$ 有 $n$ 个元素，则子集数 $2^n$，真子集数 $2^n-1$，非空真子集 $2^n-2$。
- **运算**：交 $A\cap B$、并 $A\cup B$、补 $\complement_U A$。德·摩根：$\complement_U(A\cap B)=(\complement_U A)\cup(\complement_U B)$。
- **命题与量词**：全称命题 $\forall x\in M, p(x)$；存在命题 $\exists x\in M, p(x)$。否定：$\forall$ 与 $\exists$ 互换，结论取反。
- **充分必要条件**：$p\Rightarrow q$ 称 $p$ 是 $q$ 的充分条件，$q$ 是 $p$ 的必要条件。
- **含参集合**：注意空集 $\varnothing$ 是任何集合的子集。

### 【高频考点】
1. **集合运算 + 解不等式**：常给 $A=\{x\mid x^2-3x+2\le 0\}$ 等，先化简再求交并补。
2. **含参子集问题（易漏空集）**：$A\subseteq B$ 且 $A=\{x\mid ax-1=0\}$ 时，要讨论 $A=\varnothing$（即 $a=0$ 时方程无解）的情况。
3. **充要条件判断**：结合函数、不等式、三角，区分"充分不必要 / 必要不充分 / 充要 / 既不充分也不必要"。

## 二、函数

### 知识骨架
- **三要素**：定义域、值域、对应法则。两个函数相同 ⇔ 定义域与对应法则都相同。
- **单调性**：定义法（取值—作差—变形—定号—结论）；复合函数"同增异减"。
- **奇偶性**：定义域关于原点对称是前提。奇函数 $f(-x)=-f(x)$，偶函数 $f(-x)=f(x)$。**奇函数若在 $x=0$ 有定义，则 $f(0)=0$**。
- **周期性**：$f(x+T)=f(x)$。常见推论：$f(x+a)=-f(x)\Rightarrow T=2a$；$f(x+a)=\dfrac{1}{f(x)}\Rightarrow T=2a$。
- **对称性**：$f(a+x)=f(b-x)\Leftrightarrow$ 图像关于 $x=\dfrac{a+b}{2}$ 对称。
- **基本初等函数**：指数、对数、幂函数。$a^x\cdot a^y=a^{x+y}$，$\log_a MN=\log_a M+\log_a N$，换底 $\log_a b=\dfrac{\ln b}{\ln a}$。
- **二次函数**：$y=ax^2+bx+c=a(x-h)^2+k$，顶点 $(-\dfrac{b}{2a}, \dfrac{4ac-b^2}{4a})$。

### 【高频考点】
1. **分段函数**：求值、解方程、判断单调/奇偶，注意分段点衔接。
2. **函数性质综合**（小压轴）：奇偶+单调+周期结合，例如"已知 $f(x)$ 是奇函数且周期 4，$f(1)=2$，求 $f(7)$"。
3. **指数对数比较大小**：借助中间量 0、1，或构造同底单调函数。
4. **二次函数在区间上的最值（轴动区间定 / 轴定区间动）**：必须画图分类讨论对称轴与区间的位置关系。
5. **抽象函数**：赋值法（令 $x=0,1,-1$ 等）求特殊值。

## 三、导数及其应用

### 知识骨架
- **导数定义**：$f'(x_0)=\lim\limits_{\Delta x\to 0}\dfrac{f(x_0+\Delta x)-f(x_0)}{\Delta x}$。
- **几何意义**：切线斜率。切线方程 $y-f(x_0)=f'(x_0)(x-x_0)$。
- **运算法则**：$(uv)'=u'v+uv'$，$(\dfrac{u}{v})'=\dfrac{u'v-uv'}{v^2}$。
- **常见公式**：$(e^x)'=e^x$，$(\ln x)'=\dfrac{1}{x}$，$(\sin x)'=\cos x$，$(\cos x)'=-\sin x$，$(x^n)'=nx^{n-1}$。
- **单调性**：$f'(x)>0$ 增，$f'(x)<0$ 减。
- **极值**：$f'(x_0)=0$ 且左右 $f'$ 变号。
- **最值**：极值点与端点值比较。
- **三次函数**：$f(x)=ax^3+bx^2+cx+d$，$f'(x)=3ax^2+2bx+c$ 为二次式，常考其图象与单调性。

### 【高频考点·压轴主力】
1. **切线方程**：注意"在某点处切线"（点在曲线上）与"过某点切线"（点未必在曲线上，需设切点）的区别。
2. **含参单调区间分类讨论**：解 $f'(x)\ge 0$，对参数引起的判别式符号、根的大小分类。
3. **极值与零点（方程根）个数**：分离参数 $a=g(x)$，画 $y=g(x)$ 图象数交点；或用"零点存在性定理 + 单调性"。
4. **证明不等式**：
   - 构造函数 $F(x)=f(x)-g(x)$，求 $F'(x)$ 判断符号；
   - 常用不等式：$\ln x\le x-1$，$e^x\ge x+1$，$\sin x<x<\tan x(0<x<\dfrac{\pi}{2})$。
5. **恒成立/存在性问题**：$\forall x\in D, f(x)>a$ 恒成立 ⇔ $a< f(x)_{\min}$；$\exists x$ 使 $f(x)>a$ ⇔ $a< f(x)_{\max}$。**优先分离参数**，分离不了再分类讨论。
6. **双零点 / 极值点偏移**（难题）：设 $x_1<x_2$ 为 $f(x)=t$ 的两根，常证 $x_1+x_2>2x_0$（$x_0$ 为极值点）。套路：构造对称函数 $F(x)=f(x)-f(2x_0-x)$，用单调性比较。

## 四、三角函数与解三角形

### 知识骨架
- **定义**：单位圆/坐标定义。$\sin\alpha=\dfrac{y}{r}$，$\cos\alpha=\dfrac{x}{r}$，$\tan\alpha=\dfrac{y}{x}$。
- **同角关系**：$\sin^2\alpha+\cos^2\alpha=1$，$\dfrac{\sin\alpha}{\cos\alpha}=\tan\alpha$。
- **诱导公式**："奇变偶不变，符号看象限"。
- **图像与性质**：$y=A\sin(\omega x+\varphi)$，周期 $T=\dfrac{2\pi}{|\omega|}$，频率 $f=\dfrac{1}{T}$，初相 $\varphi$。由最值求 $A$，由周期求 $\omega$，由最值点求 $\varphi$。
- **和差倍半公式**：
  - $\sin(\alpha\pm\beta)=\sin\alpha\cos\beta\pm\cos\alpha\sin\beta$
  - $\cos(\alpha\pm\beta)=\cos\alpha\cos\beta\mp\sin\alpha\sin\beta$
  - $\sin 2\alpha=2\sin\alpha\cos\alpha$，$\cos 2\alpha=\cos^2\alpha-\sin^2\alpha=2\cos^2\alpha-1=1-2\sin^2\alpha$
  - 降幂：$\cos^2\alpha=\dfrac{1+\cos 2\alpha}{2}$，$\sin^2\alpha=\dfrac{1-\cos 2\alpha}{2}$
- **辅助角公式**：$a\sin x+b\cos x=\sqrt{a^2+b^2}\sin(x+\varphi)$，$\tan\varphi=\dfrac{b}{a}$。
- **解三角形**：
  - 正弦定理 $\dfrac{a}{\sin A}=\dfrac{b}{\sin B}=\dfrac{c}{\sin C}=2R$
  - 余弦定理 $a^2=b^2+c^2-2bc\cos A$
  - 面积 $S=\dfrac{1}{2}ab\sin C=\dfrac{abc}{4R}$
  - 射影定理 $a=b\cos C+c\cos B$

### 【高频考点】
1. **三角恒等变换求值**：给角求值、给值求值、给值求角。注意角的范围（定符号）。
2. **$y=A\sin(\omega x+\varphi)$ 图像变换**：平移 $\varphi$ 与伸缩 $\omega$ 的**顺序**（先平移后伸缩 vs 先伸缩后平移，平移量不同）。
3. **解三角形综合（大题第一问常客）**：边角互化（遇 $a^2+b^2-c^2$ 想余弦定理，遇 $a\sin B=b\sin A$ 想正弦定理），判断三角形形状，求面积/周长最值。
4. **范围问题**：已知一角和一边，求第三边范围，常结合"$A+B+C=\pi$ + 基本不等式"或正弦定理化为单一角的函数。

## 五、平面向量

### 知识骨架
- **线性运算**：加减法（三角形法则/平行四边形法则）、数乘。
- **共线定理**：$\vec a//\vec b(\vec b\neq\vec 0)\Leftrightarrow \vec a=\lambda\vec b\Leftrightarrow x_1y_2-x_2y_1=0$。
- **数量积（点积）**：$\vec a\cdot\vec b=|\vec a||\vec b|\cos\theta=x_1x_2+y_1y_2$。
  - 模：$|\vec a|=\sqrt{x^2+y^2}$
  - 夹角：$\cos\theta=\dfrac{\vec a\cdot\vec b}{|\vec a||\vec b|}$
  - 垂直：$\vec a\perp\vec b\Leftrightarrow\vec a\cdot\vec b=0\Leftrightarrow x_1x_2+y_1y_2=0$
- **基底与坐标**：平面向量基本定理。
- **重要结论**：
  - $\vec a\cdot\vec b\le|\vec a||\vec b|$（当且仅当同向取等）
  - $|\vec a+\vec b|^2+|\vec a-\vec b|^2=2(|\vec a|^2+|\vec b|^2)$（平行四边形对角线定理）

### 【高频考点】
1. **数量积求夹角/模**：坐标法最稳。
2. **向量与几何图形结合**：在三角形/四边形中给定向量关系，常用**建系坐标化**或**选基底**（如以两边为基底表示第三边）。
3. **求最值（模的范围）**：常常化为 $|\vec a-\vec b|$ 几何意义（两点距离），或用数量积不等式。
4. **向量在三角/函数中的渗透**：辅助角公式的向量背景。

## 六、数列

### 知识骨架
- **概念**：$a_n$ 与 $S_n$ 关系：$a_n=\begin{cases}S_1, & n=1\\ S_n-S_{n-1}, & n\ge2\end{cases}$（**务必验证 $n=1$**）。
- **等差数列**：
  - 通项 $a_n=a_1+(n-1)d$
  - 前 $n$ 项和 $S_n=\dfrac{n(a_1+a_n)}{2}=na_1+\dfrac{n(n-1)}{2}d$
  - 性质：$a_m+a_n=a_p+a_q\Leftrightarrow m+n=p+q$；$S_{2n}=\dfrac{2n}{2}(a_1+a_{2n})$ 等。
- **等比数列**：
  - 通项 $a_n=a_1q^{n-1}$
  - 前 $n$ 项和 $S_n=\begin{cases}na_1, & q=1\\ \dfrac{a_1(1-q^n)}{1-q}, & q\ne1\end{cases}$
  - 性质：$a_m\cdot a_n=a_p\cdot a_q\Leftrightarrow m+n=p+q$。
- **求和方法**：
  - 公式法
  - **错位相减法**：$a_n=b_n\cdot c_n$（等差×等比），如 $a_n=(2n-1)\cdot 2^n$。
  - **裂项相消法**：$a_n=\dfrac{1}{n(n+1)}=\dfrac{1}{n}-\dfrac{1}{n+1}$，通项 $\dfrac{1}{n(n+k)}=\dfrac{1}{k}(\dfrac{1}{n}-\dfrac{1}{n+k})$。
  - **分组求和法**：等差+等比混合，如 $a_n=2n+\dfrac{1}{2^n}$。
  - **并项求和**：符号交替 $a_n=(-1)^n f(n)$。
  - **倒序相加**：用于"首末等距项之和相等"。

### 【高频考点】
1. **由 $S_n$ 求 $a_n$（易错点 $n=1$）**：必考小题。
2. **等差/等比基本量运算**：知三求二，方程思想。
3. **错位相减法（大题标准题型）**：写出 $S_n$ 与 $qS_n$，相减后中间项成等比，**注意最后一项处理和化简**，最后用 $S_n$ 验证 $n=1,2$ 防错。
4. **裂项相消**：注意消去哪些项、保留哪些项，写出前几项和最后几项看清结构。
5. **数列与不等式综合（压轴）**：证明 $\sum a_n$ 范围或与某常数比较，常配合放缩（$\dfrac{1}{n^2}<\dfrac{1}{n(n-1)}$ 等）。

## 七、不等式

### 知识骨架
- **基本不等式**：$a+b\ge 2\sqrt{ab}$（$a,b>0$，当且仅当 $a=b$ 取等）。
  - 变式：$\dfrac{a^2+b^2}{2}\ge(\dfrac{a+b}{2})^2\ge ab$，$ab\le(\dfrac{a+b}{2})^2$。
- **一元二次不等式**：借助二次函数图象，先看开口、再看判别式、最后定根。
- **简单线性规划**（部分省份已淡化）：画可行域，目标函数平移求最值。
- **绝对值不等式**：$|a|-|b|\le|a\pm b|\le|a|+|b|$。

### 【高频考点】
1. **基本不等式求最值（"1 的代换"）**：如 $y=x+\dfrac{4}{x}(x>0)$，或 $y=\dfrac{2}{x}+\dfrac{x}{2}$ 凑积定值。**三要素：一正、二定、三相等**，缺一不可。
2. **二次不等式恒成立**：$ax^2+bx+c>0$ 恒成立 ⇔ $a>0$ 且 $\Delta<0$。
3. **不等式与实际应用题**：造价最低、用料最少，建目标函数后用基本不等式或导数。

## 八、立体几何

### 知识骨架
- **空间位置关系**：线线、线面、面面的平行与垂直。
  - 线面平行：$a\not\subset\alpha, b\subset\alpha, a//b\Rightarrow a//\alpha$。
  - 面面平行 ⇔ 一个面内两相交直线平行于另一面。
  - 线面垂直：$l\perp\alpha\Leftrightarrow l\perp$ 面内两相交直线。
- **空间角**：线线角、线面角（$\sin\theta=\dfrac{|h|}{|l|}$，找射影）、二面角。
- **空间距离**：点到面距离（等体积法 $V=\dfrac{1}{3}Sh$ 求高）。
- **几何体**：柱、锥、台、球。体积 $V_{\text{柱}}=Sh$，$V_{\text{锥}}=\dfrac{1}{3}Sh$，$V_{\text{球}}=\dfrac{4}{3}\pi R^3$，表面积 $S_{\text{球}}=4\pi R^2$。
- **外接球/内切球**：
  - 长方体外接球 $R=\dfrac{\sqrt{a^2+b^2+c^2}}{2}$。
  - 正棱锥外接球：构造直角三角形 $R^2=(h-R)^2+r^2$（$h$ 为锥高，$r$ 为底面外接圆半径）。

### 【高频考点】
1. **外接球半径（小题压轴常客）**：补形法（补成长方体）、构造法（轴截面直角三角形）。**记套路：正三棱锥/正四棱锥/侧棱垂直底面三类公式**。
2. **空间向量法证平行垂直 + 求角（大题标准解法）**：
   - 建系，写点坐标；
   - 求面法向量 $\vec n$（设 $\vec n=(x,y,z)$，由 $\vec n\cdot\vec a=0,\vec n\cdot\vec b=0$ 解出）；
   - 线面角 $\theta$：$\sin\theta=|\cos\langle\vec l,\vec n\rangle|$；
   - 二面角：两法向量夹角，**注意锐钝角的判断（看图）**。
3. **翻折/折叠问题**：折前折后哪些量不变（共线点的距离、同侧角），哪些变化。
4. **体积最值**：动点在棱上滑动，建参数，用导数或基本不等式求最值。

## 九、解析几何（平面解析几何）

### 知识骨架
- **直线**：点斜式 $y-y_0=k(x-x_0)$；斜率 $k=\dfrac{y_2-y_1}{x_2-x_1}$；点到线距离 $d=\dfrac{|Ax_0+By_0+C|}{\sqrt{A^2+B^2}}$；两平行线距离 $d=\dfrac{|C_1-C_2|}{\sqrt{A^2+B^2}}$。
- **圆**：$(x-a)^2+(y-b)^2=r^2$；圆心到直线距离与 $r$ 比较判位置关系。
- **椭圆**：$\dfrac{x^2}{a^2}+\dfrac{y^2}{b^2}=1(a>b>0)$，$c^2=a^2-b^2$，离心率 $e=\dfrac{c}{a}\in(0,1)$。
- **双曲线**：$\dfrac{x^2}{a^2}-\dfrac{y^2}{b^2}=1(a,b>0)$，$c^2=a^2+b^2$，$e=\dfrac{c}{a}>1$，渐近线 $y=\pm\dfrac{b}{a}x$。
- **抛物线**：$y^2=2px$，焦点 $(\dfrac{p}{2},0)$，准线 $x=-\dfrac{p}{2}$，焦半径 $|PF|=x_0+\dfrac{p}{2}$。

### 【高频考点·压轴主力】
1. **离心率范围**：由几何条件（如焦点三角形、准线关系）列 $a,b,c$ 不等式，化为 $e$ 的范围。
2. **直线与圆锥曲线位置关系（大题核心）**：
   - 设直线 $l$（注意斜率是否存在，必要时设 $x=my+t$ 避免讨论）；
   - 与曲线方程联立，得 $Ax^2+Bx+C=0$，**必写 $\Delta>0$ 且韦达定理 $x_1+x_2, x_1x_2$**；
   - 翻译条件（弦长、面积、向量、斜率关系）为 $x_1,x_2$ 表达式，代入韦达化简。
3. **弦长公式**：$|AB|=\sqrt{1+k^2}|x_1-x_2|=\sqrt{1+k^2}\sqrt{(x_1+x_2)^2-4x_1x_2}$。
4. **面积问题**：$\triangle FAB$ 面积常用 $S=\dfrac{1}{2}|AB|\cdot d$（$d$ 为另一焦点到直线距离）或 $\dfrac{1}{2}|y_1-y_2|·|x_F|$。
5. **定点定值问题**：设参数，化简后令其与参数无关即得定点/定值。**设而不求**是核心思想。
6. **轨迹方程**：直接法、定义法、相关点法（代入法）、参数法、交轨法。
7. **抛物线焦点弦性质**（小题常用）：$AB$ 过焦点，$x_1x_2=\dfrac{p^2}{4}$，$y_1y_2=-p^2$，$|AB|=x_1+x_2+p$，$\dfrac{1}{|AF|}+\dfrac{1}{|BF|}=\dfrac{2}{p}$。

## 十、计数原理与概率统计

### 知识骨架
- **两个原理**：分类加法、分步乘法。
- **排列组合**：$A_n^m=\dfrac{n!}{(n-m)!}$，$C_n^m=\dfrac{A_n^m}{m!}$。组合性质 $C_n^m=C_n^{n-m}$。
- **二项式定理**：$(a+b)^n=\sum\limits_{k=0}^n C_n^k a^{n-k}b^k$。通项 $T_{r+1}=C_n^r a^{n-r}b^r$。
- **概率**：
  - 古典概型 $P=\dfrac{m}{n}$；
  - 互斥 $P(A\cup B)=P(A)+P(B)$；独立 $P(AB)=P(A)P(B)$；
  - 条件概率 $P(B|A)=\dfrac{P(AB)}{P(A)}$。
- **随机变量**：
  - 分布列、期望 $E(X)=\sum x_i p_i$、方差 $D(X)=\sum(x_i-E(X))^2p_i=E(X^2)-E^2(X)$。
  - 二项分布 $X\sim B(n,p)$：$E=np,D=np(1-p)$。
  - 超几何分布；正态分布 $N(\mu,\sigma^2)$（$3\sigma$ 原则）。
- **统计**：抽样（简单随机、分层、系统）；样本均值/方差；线性回归 $\hat y=\hat bx+\hat a$，$\hat b=\dfrac{\sum(x_i-\bar x)(y_i-\bar y)}{\sum(x_i-\bar x)^2}$；独立性检验 $\chi^2$。

### 【高频考点】
1. **二项式定理指定项系数**：用通项 $T_{r+1}$，求特定 $x$ 次幂对应 $r$；注意"系数"与"二项式系数 $C_n^r$"区别；最大系数用不等式 $\dfrac{C_n^{r-1}}{C_n^r}\le 1\le\dfrac{C_n^r}{C_n^{r+1}}$。
2. **排列组合应用（易错）**：捆绑法（相邻）、插空法（不相邻）、间接法（总数-不满足）、隔板法（相同元素分组）。注意"至少""恰好"的转化。
3. **条件概率与全概率公式**（新增重点）：结合具体情境（如检测、传递信息）。
4. **离散型随机变量分布列与期望（大题）**：
   - 确定变量取值；
   - 逐一计算概率（写清所用模型：二项/超几何/独立重复）；
   - 列分布列、求 $E,D$。
5. **正态分布对称性**：$P(X<\mu)=0.5$，$P(X<\mu-\sigma)=P(X>\mu+\sigma)$，利用对称求区间概率。
6. **回归方程预测 + 残差 / $\chi^2$ 独立性检验**：套公式，注意临界值比较给出结论（犯错误概率表述）。

## 十一、复数

### 知识骨架
- **概念**：$z=a+bi$（$a$ 实部、$b$ 虚部），$i^2=-1$。
- **几何意义**：复平面内点 $(a,b)$，模 $|z|=\sqrt{a^2+b^2}$，共轭 $\bar z=a-bi$。
- **运算**：加减乘除（除法分子分母同乘分母共轭）。
- **重要结论**：$|z|^2=z\cdot\bar z$；$|z_1z_2|=|z_1||z_2|$；$\dfrac{z_1}{z_2}$ 模为 $\dfrac{|z_1|}{|z_2|}$。

### 【高频考点】
1. **复数四则运算 + 几何意义**：求对应点所在象限、模。
2. **纯虚数条件**：实部为 0 且虚部不为 0（易漏"虚部不为 0"）。

## 十二、选修与拓展（视地区）

### 知识骨架
- **极坐标与参数方程**：
  - 直角坐标与极坐标互化 $x=\rho\cos\theta, y=\rho\sin\theta$；
  - 圆的极坐标、直线的参数方程 $\begin{cases}x=x_0+t\cos\alpha\\y=y_0+t\sin\alpha\end{cases}$（$t$ 几何意义为有向距离，弦长 $|t_1-t_2|$）；
  - 椭圆参数方程 $\begin{cases}x=a\cos\varphi\\y=b\sin\varphi\end{cases}$。
- **不等式选讲**：绝对值不等式 $|f(x)|-|g(x)|\le a$ 解集与最值；柯西不等式 $(\sum a_i^2)(\sum b_i^2)\ge(\sum a_ib_i)^2$。

### 【高频考点】
1. **参数方程中弦长**：用直线参数方程 $t$ 的几何意义，$|AB|=|t_1-t_2|$，避免联立复杂运算。
2. **极坐标方程化为直角坐标**后求交点/距离。
3. **绝对值函数最值**：零点分区间法去绝对值，分段求最值。
