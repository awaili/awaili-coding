# 模块六 · 数列 · 重点公式与推导

> 本文件对应广东高中数学（人教A版2019、新高考Ⅰ卷）"数列"模块。数列是新高考解答题常考模块（通常第19、20题位置），考查通项求法与前 n 项和求法，并强调与函数、不等式的综合。本文档把该模块所有重点公式与结论逐一给出**完整推导**，每步注明依据，便于复习时直接背诵推导链条。

## 一、通项 $a_n$ 与前 n 项和 $S_n$ 的关系

### 1.1 基本关系式

**公式**：
$$a_n=\begin{cases}S_1, & n=1,\\[4pt] S_n-S_{n-1}, & n\ge 2.\end{cases}$$

**推导**：
1. 前n项和定义：$S_n=a_1+a_2+\cdots+a_n$（依据前 n 项和定义）。
2. 同理 $S_{n-1}=a_1+a_2+\cdots+a_{n-1}$（$n\ge 2$）。
3. 两式相减（依据减法）：
   $$S_n-S_{n-1}=(a_1+a_2+\cdots+a_n)-(a_1+a_2+\cdots+a_{n-1})=a_n.$$
4. 当 $n=1$ 时 $S_{n-1}=S_0$ 无定义（数列从 $a_1$ 起），故 $n=1$ 单独处理：$a_1=S_1$。

**关键提醒**：用 $a_n=S_n-S_{n-1}$ 求通项后，**必须**验证 $n=1$ 时公式是否给出 $a_1=S_1$：
- 若 $n=1$ 代入公式所得值等于 $S_1$，则统一用一个表达式；
- 若不等，则通项写成分段形式 $\begin{cases}S_1,&n=1\\S_n-S_{n-1},&n\ge 2\end{cases}$。

### 1.2 典型应用

**示例**：设 $S_n=n^2+3n$，求 $a_n$。
1. $n=1$：$a_1=S_1=1+3=4$。
2. $n\ge 2$：$a_n=S_n-S_{n-1}=(n^2+3n)-\big[(n-1)^2+3(n-1)\big]=n^2+3n-(n^2-2n+1+3n-3)=n^2+3n-(n^2+n-2)=2n+2$。
3. 验证 $n=1$：$2\cdot1+2=4=a_1$，吻合。
4. 故 $a_n=2n+2$（统一表达式）。

**反例**：设 $S_n=2^n+1$，求 $a_n$。
1. $n=1$：$a_1=S_1=3$。
2. $n\ge 2$：$a_n=2^n+1-(2^{n-1}+1)=2^n-2^{n-1}=2^{n-1}$。
3. 验证 $n=1$：$2^{1-1}=1\ne 3$，不吻合。
4. 故 $a_n=\begin{cases}3,&n=1\\2^{n-1},&n\ge 2\end{cases}$（分段形式）。

## 二、等差数列

### 2.1 通项公式 $a_n=a_1+(n-1)d$

**公式**：
$$a_n=a_1+(n-1)d.$$

**推导**（累加法）：
1. 等差数列定义：$a_{n+1}-a_n=d$（$n\ge 1$，$d$ 为公差）（依据定义）。
2. 列出 $n-1$ 个等式（$n\ge 2$）：
   $$a_2-a_1=d,\quad a_3-a_2=d,\quad \ldots,\quad a_n-a_{n-1}=d.$$
3. 左右分别相加（依据等式同向相加）：
   $$(a_2-a_1)+(a_3-a_2)+\cdots+(a_n-a_{n-1})=(n-1)d.$$
4. 左边抵消（依据裂项相消，中间项两两抵消）：
   $$-a_1+a_n=(n-1)d.$$
5. 移项得 $a_n=a_1+(n-1)d$。

**说明**：$a_n=a_1+(n-1)d=d\cdot n+(a_1-d)$，即 $a_n$ 是 $n$ 的一次函数（$d\ne 0$）或常数（$d=0$）。由此可知等差数列的点 $(n,a_n)$ 共线。

---

### 2.2 前 n 项和公式 $S_n=\dfrac{n(a_1+a_n)}{2}=na_1+\dfrac{n(n-1)}{2}d$

**公式**：
$$S_n=\dfrac{n(a_1+a_n)}{2}=na_1+\dfrac{n(n-1)}{2}d.$$

**推导**（倒序相加法）：
1. 写出 $S_n$：
   $$S_n=a_1+a_2+\cdots+a_{n-1}+a_n.\quad(*)$$
2. 倒序写出 $S_n$：
   $$S_n=a_n+a_{n-1}+\cdots+a_2+a_1.\quad(**)$$
3. 两式相加（依据等式相加）：
   $$2S_n=(a_1+a_n)+(a_2+a_{n-1})+\cdots+(a_n+a_1).$$
4. 等差数列性质：$a_k+a_{n+1-k}=a_1+(k-1)d+a_1+(n-k)d=2a_1+(n-1)d$（依据 2.1 通项），与 $k$ 无关，即每对和都等于 $a_1+a_n$。
5. 共 $n$ 对，故 $2S_n=n(a_1+a_n)$，即 $S_n=\dfrac{n(a_1+a_n)}{2}$。
6. 代入 $a_n=a_1+(n-1)d$：
   $$S_n=\dfrac{n}{2}\big[2a_1+(n-1)d\big]=na_1+\dfrac{n(n-1)}{2}d.$$

**说明**：$S_n=\dfrac{d}{2}n^2+\big(a_1-\dfrac{d}{2}\big)n$，是关于 $n$ 的二次函数（$d\ne 0$），常数项为 0。故 $(n,S_n)$ 共线于抛物线 $y=\dfrac d2 x^2+\big(a_1-\dfrac d2\big)x$。

---

### 2.3 性质 $a_m+a_n=a_p+a_q$（$m+n=p+q$）

**公式**：在等差数列 $\{a_n\}$ 中，若 $m+n=p+q$（$m,n,p,q\in\mathbb{N}^*$），则
$$a_m+a_n=a_p+a_q.$$

**推导**（用通项）：
1. 由通项公式 2.1：
   $$a_m=a_1+(m-1)d,\quad a_n=a_1+(n-1)d.$$
2. 相加：
   $$a_m+a_n=2a_1+(m+n-2)d.\quad(*)$$
3. 同理：
   $$a_p+a_q=2a_1+(p+q-2)d.\quad(**)$$
4. 由 $m+n=p+q$（依据条件），得 $(m+n-2)d=(p+q-2)d$。
5. 故 $a_m+a_n=a_p+a_q$。

**特例**：$m+n=2k$ 时 $a_m+a_n=2a_k$，即等差数列中"等距离三项成等差"：$a_{k-r},a_k,a_{k+r}$ 满足 $2a_k=a_{k-r}+a_{k+r}$。

**推论**：等差数列中若 $m+n=p+q$，则 $S$ 中下标和相等两两配对和相等，常用于简化 $S_n$ 的计算。

---

### 2.4 等差数列前 n 项和的性质

**性质1**：等差数列 $\{a_n\}$ 中，$S_m,S_{2m}-S_m,S_{3m}-S_{2m},\ldots$ 仍成等差数列，公差为 $m^2 d$。

**推导**：
1. $S_m=ma_1+\dfrac{m(m-1)}{2}d$。
2. $S_{2m}-S_m=\sum\limits_{k=m+1}^{2m}a_k$，即第 $m+1$ 到 $2m$ 项之和。此段首项 $a_{m+1}=a_1+md$，末项 $a_{2m}=a_1+(2m-1)d$，和为 $m\cdot\dfrac{a_{m+1}+a_{2m}}{2}=m\big[a_1+\dfrac{(m+2m-1)d}{2}\big]=m a_1+\dfrac{m(3m-1)}{2}d$。
3. $(S_{2m}-S_m)-S_m=\dfrac{m(3m-1)}{2}d-\dfrac{m(m-1)}{2}d=\dfrac{m(3m-1-m+1)}{2}d=\dfrac{2m^2}{2}d=m^2 d$。
4. 同理 $(S_{3m}-S_{2m})-(S_{2m}-S_m)=m^2 d$，故这三段成等差数列，公差 $m^2 d$。

**性质2**：若 $S_n$ 为等差数列前 n 项和，则 $\dfrac{S_n}{n}=\dfrac{a_1+a_n}{2}=a_{\frac{n+1}{2}}$（$n$ 奇数时为中间项）。

**推导**：$S_n=\dfrac{n(a_1+a_n)}{2}$，故 $\dfrac{S_n}{n}=\dfrac{a_1+a_n}{2}$，即前 n 项平均数等于首末项平均数。当 $n=2k-1$ 奇数时，$a_1+a_{2k-1}=2a_k$（依据 2.3 性质，$1+(2k-1)=2k$），故 $\dfrac{S_n}{n}=a_k=a_{\frac{n+1}{2}}$。

## 三、等比数列

### 3.1 通项公式 $a_n=a_1 q^{n-1}$

**公式**：
$$a_n=a_1 q^{n-1}\quad(q\ne 0).$$

**推导**（累乘法）：
1. 等比数列定义：$\dfrac{a_{n+1}}{a_n}=q$（$n\ge 1$，$q$ 为公比，$q\ne 0$）（依据定义）。
2. 列出 $n-1$ 个等式（$n\ge 2$）：
   $$\dfrac{a_2}{a_1}=q,\quad \dfrac{a_3}{a_2}=q,\quad \ldots,\quad \dfrac{a_n}{a_{n-1}}=q.$$
3. 左右分别相乘（依据等式同向相乘）：
   $$\dfrac{a_2}{a_1}\cdot\dfrac{a_3}{a_2}\cdots\dfrac{a_n}{a_{n-1}}=q^{n-1}.$$
4. 左边约分（依据分子分母相消，中间项两两抵消）：
   $$\dfrac{a_n}{a_1}=q^{n-1}.$$
5. 两边乘 $a_1$ 得 $a_n=a_1 q^{n-1}$。

**说明**：$a_n=a_1 q^{n-1}$，当 $q>0$ 时是指数型函数；$q=1$ 时为常数列；$q<0$ 时为摆动数列。$q=0$ 不构成等比数列（除 $a_1$ 外各项为 0，比值无定义）。

---

### 3.2 前 n 项和公式 $S_n=\dfrac{a_1(1-q^n)}{1-q}$（$q\ne 1$）

**公式**：
$$S_n=\begin{cases}na_1, & q=1,\\[4pt] \dfrac{a_1(1-q^n)}{1-q}=\dfrac{a_1(q^n-1)}{q-1}, & q\ne 1.\end{cases}$$

**推导**（错位相减法，重点写清 $q\ne 1$ 与 $q=1$）：

**情形一：$q\ne 1$。**
1. 写出 $S_n$：
   $$S_n=a_1+a_1q+a_1q^2+\cdots+a_1q^{n-1}.\quad(*)$$
2. 两边同乘 $q$（依据等式两边同乘非零数）：
   $$qS_n=a_1q+a_1q^2+a_1q^3+\cdots+a_1q^n.\quad(**)$$
3. $(*)-(**)$（依据等式相减；注意左边减左边、右边项按对应位置相减）：
   $$S_n-qS_n=a_1+\big(a_1q-a_1q\big)+\big(a_1q^2-a_1q^2\big)+\cdots+\big(a_1q^{n-1}-a_1q^{n-1}\big)-a_1q^n.$$
4. 中间项全部抵消（依据错位相减），得
   $$(1-q)S_n=a_1-a_1q^n=a_1(1-q^n).$$
5. 因 $q\ne 1$，$1-q\ne 0$，两边除以 $1-q$：
   $$S_n=\dfrac{a_1(1-q^n)}{1-q}.$$

**情形二：$q=1$。**
1. 此时 $a_n=a_1$ 对所有 $n$ 成立（依据 3.1）。
2. $S_n=\underbrace{a_1+a_1+\cdots+a_1}_{n\text{个}}=na_1$。

**关键提醒**：使用前 n 项和公式**必须先讨论 $q=1$ 与 $q\ne 1$**，这是高考解答题高频失分点。

**等价形式**：当 $q\ne 1$ 时 $S_n=\dfrac{a_1(q^n-1)}{q-1}=\dfrac{a_1-a_n q}{1-q}$（代入 $a_n=a_1 q^{n-1}$，$a_1 q^n=a_n q$）。

---

### 3.3 性质 $a_m\cdot a_n=a_p\cdot a_q$（$m+n=p+q$）

**公式**：在等比数列 $\{a_n\}$ 中，若 $m+n=p+q$（$m,n,p,q\in\mathbb{N}^*$），则
$$a_m\cdot a_n=a_p\cdot a_q.$$

**推导**（用通项）：
1. 由 3.1：$a_m=a_1 q^{m-1}$，$a_n=a_1 q^{n-1}$。
2. 相乘：
   $$a_m\cdot a_n=a_1^2 q^{m+n-2}.\quad(*)$$
3. 同理 $a_p\cdot a_q=a_1^2 q^{p+q-2}.\quad(**)$
4. 由 $m+n=p+q$（依据条件），$q^{m+n-2}=q^{p+q-2}$。
5. 故 $a_m\cdot a_n=a_p\cdot a_q$。

**特例**：$m+n=2k$ 时 $a_m a_n=a_k^2$，即等比数列中"等距离三项成等比"：$a_{k-r},a_k,a_{k+r}$ 满足 $a_k^2=a_{k-r}\cdot a_{k+r}$（各项同号时）。

---

### 3.4 等比数列前 n 项和的性质

**性质**：等比数列 $\{a_n\}$（$q\ne 1$）的前 n 项和 $S_n$ 满足 $S_n,S_{2n}-S_n,S_{3n}-S_{2n},\ldots$ 仍成等比数列，公比为 $q^n$。

**推导**：
1. $S_n=a_1\dfrac{1-q^n}{1-q}$。
2. $S_{2n}-S_n=a_1\dfrac{1-q^{2n}}{1-q}-a_1\dfrac{1-q^n}{1-q}=a_1\dfrac{q^n-q^{2n}}{1-q}=a_1\dfrac{q^n(1-q^n)}{1-q}=q^n\cdot S_n$。
3. 同理 $S_{3n}-S_{2n}=q^n(S_{2n}-S_n)=q^{2n}S_n$。
4. 故 $S_n,S_{2n}-S_n,S_{3n}-S_{2n},\ldots$ 是以 $S_n$ 为首项、$q^n$ 为公比的等比数列。

---

### 3.5 等比数列通项的其它表达

**公式**：$a_n=a_m q^{n-m}$（$m\le n$）。

**推导**：
1. 由 3.1：$a_n=a_1 q^{n-1}$，$a_m=a_1 q^{m-1}$。
2. 相除：$\dfrac{a_n}{a_m}=\dfrac{a_1 q^{n-1}}{a_1 q^{m-1}}=q^{n-m}$。
3. 故 $a_n=a_m q^{n-m}$。这给出"任意一项作为基准"的通项写法。

**示例**：已知等比数列 $a_3=8$，$a_5=32$，求 $a_n$。由 $\dfrac{a_5}{a_3}=q^2=4$，$q=\pm 2$；$a_n=a_3 q^{n-3}=8(\pm 2)^{n-3}$。若 $q=2$，$a_n=2^n$；若 $q=-2$，$a_n=(-2)^{n-1}$（结合 $a_3=8=(-2)^2\cdot2$ 验证 $q=-2$ 时 $a_3=(-2)^2\cdot a_1=4a_1$，$a_1=2$，$a_n=2(-2)^{n-1}$）。

## 四、求和常用方法

### 4.1 裂项相消法：$\dfrac{1}{n(n+1)}=\dfrac{1}{n}-\dfrac{1}{n+1}$

**公式**：
$$\dfrac{1}{n(n+1)}=\dfrac{1}{n}-\dfrac{1}{n+1}.$$

**推导**（待定系数法）：
1. 设 $\dfrac{1}{n(n+1)}=\dfrac{A}{n}+\dfrac{B}{n+1}$，其中 $A,B$ 待定。
2. 右边通分（依据分式通分）：$\dfrac{A}{n}+\dfrac{B}{n+1}=\dfrac{A(n+1)+Bn}{n(n+1)}=\dfrac{(A+B)n+A}{n(n+1)}$。
3. 与左边 $\dfrac{1}{n(n+1)}$ 比较分子（依据分式相等多项式恒等）：
   $$(A+B)n+A=1.$$
4. 对所有 $n$ 成立 ⇒ $n$ 的系数为 0，常数项为 1：
   $$A+B=0,\quad A=1.$$
5. 解得 $A=1$，$B=-1$。
6. 故 $\dfrac{1}{n(n+1)}=\dfrac{1}{n}-\dfrac{1}{n+1}$。

**求和**：
$$\sum_{k=1}^{n}\dfrac{1}{k(k+1)}=\sum_{k=1}^{n}\!\left(\dfrac1k-\dfrac1{k+1}\right)=1-\dfrac{1}{n+1}=\dfrac{n}{n+1}.$$
（裂项后中间项 $\dfrac1k$ 与 $-\dfrac1k$ 相消，只剩首项 $1$ 与末项 $-\dfrac{1}{n+1}$。）

---

### 4.2 推广：$\dfrac{1}{n(n+k)}=\dfrac{1}{k}\!\left(\dfrac{1}{n}-\dfrac{1}{n+k}\right)$

**公式**（$k\in\mathbb{N}^*$）：
$$\dfrac{1}{n(n+k)}=\dfrac{1}{k}\!\left(\dfrac{1}{n}-\dfrac{1}{n+k}\right).$$

**推导**（待定系数法）：
1. 设 $\dfrac{1}{n(n+k)}=\dfrac{A}{n}+\dfrac{B}{n+k}$。
2. 通分：$\dfrac{A(n+k)+Bn}{n(n+k)}=\dfrac{(A+B)n+Ak}{n(n+k)}$。
3. 比较分子：$(A+B)n+Ak=1$，故
   $$A+B=0,\quad Ak=1.$$
4. 解得 $A=\dfrac1k$，$B=-\dfrac1k$。
5. 故 $\dfrac{1}{n(n+k)}=\dfrac{1}{k}\!\left(\dfrac{1}{n}-\dfrac{1}{n+k}\right)$。

**求和示例**：$\sum\limits_{k=1}^{n}\dfrac{1}{k(k+2)}=\dfrac12\sum\limits_{k=1}^{n}\!\left(\dfrac1k-\dfrac1{k+2}\right)$。展开后 $-\dfrac12$ 与 $+\dfrac12$ 跨两项抵消，结果为 $\dfrac12\!\left(1+\dfrac12-\dfrac{1}{n+1}-\dfrac{1}{n+2}\right)$。

---

### 4.3 裂项的其它常见形式

**形式1（根式）**：$\dfrac{1}{\sqrt{n}+\sqrt{n+1}}=\sqrt{n+1}-\sqrt{n}$。
*推导*：分子有理化 $\dfrac{1}{\sqrt{n}+\sqrt{n+1}}\cdot\dfrac{\sqrt{n+1}-\sqrt{n}}{\sqrt{n+1}-\sqrt{n}}=\dfrac{\sqrt{n+1}-\sqrt{n}}{(n+1)-n}=\sqrt{n+1}-\sqrt{n}$。

**形式2（对数）**：$\lg\dfrac{n+1}{n}=\lg(n+1)-\lg n$。
*推导*：由对数减法 $\lg\dfrac{a}{b}=\lg a-\lg b$ 直接得。

**形式3（阶乘）**：$\dfrac{n}{(n+1)!}=\dfrac{1}{n!}-\dfrac{1}{(n+1)!}$。
*推导*：$\dfrac{1}{n!}-\dfrac{1}{(n+1)!}=\dfrac{n+1-1}{(n+1)!}=\dfrac{n}{(n+1)!}$。

**形式4（指数）**：$n q^n=\dfrac{n q^n-(n+1)q^{n+1}+q^{n+1}}{1-q}$，用于裂项求等差×等比型和。常配合错位相减法使用。

---

### 4.4 错位相减法：适用条件与操作步骤

**适用条件**：数列通项 $b_n=a_n\cdot c_n$，其中 $\{a_n\}$ 为等差数列（公差 $d$）、$\{c_n\}$ 为等比数列（公比 $q\ne 1$），即 $b_n=(A+Bn)q^n$ 型。

**操作步骤**（以 $b_n=a_n\cdot q^{n-1}$ 为例，$\{a_n\}$ 等差）：
1. 写出 $S_n=b_1+b_2+\cdots+b_n$：
   $$S_n=a_1\cdot 1+a_2\cdot q+\cdots+a_n\cdot q^{n-1}.\quad(*)$$
2. 两边同乘 $q$：
   $$qS_n=a_1\cdot q+a_2\cdot q^2+\cdots+a_n\cdot q^n.\quad(**)$$
3. $(*)-(**)$，左边 $(1-q)S_n$；右边按位相减：
   $$(1-q)S_n=a_1+(a_2-a_1)q+(a_3-a_2)q^2+\cdots+(a_n-a_{n-1})q^{n-1}-a_n q^n.$$
4. 由 $\{a_n\}$ 等差，$a_k-a_{k-1}=d$，故
   $$(1-q)S_n=a_1+d(q+q^2+\cdots+q^{n-1})-a_n q^n.$$
5. 用等比求和 $q+q^2+\cdots+q^{n-1}=\dfrac{q(1-q^{n-1})}{1-q}$（$q\ne 1$），代入得
   $$(1-q)S_n=a_1+d\cdot\dfrac{q(1-q^{n-1})}{1-q}-a_n q^n.$$
6. 两边除以 $1-q$ 即得 $S_n$。

**关键提醒**：步骤 3 中相减时**第一项 $a_1$ 保留、最后一项 $-a_n q^n$ 保留、中间项系数全为 $d$**，这是错位相减法的核心结构。务必检验 $q\ne 1$（$q=1$ 时退化为等差数列求和）。

**完整示例**：求 $S_n=1\cdot 2+2\cdot 2^2+3\cdot 2^3+\cdots+n\cdot 2^n$。
1. $S_n=1\cdot 2+2\cdot 2^2+\cdots+n\cdot 2^n.\quad(*)$
2. $2S_n=1\cdot 2^2+2\cdot 2^3+\cdots+n\cdot 2^{n+1}.\quad(**)$
3. $(*)-(**)$：$S_n-2S_n=2+(2-1)2^2+(3-2)2^3+\cdots+(n-(n-1))2^n-n\cdot 2^{n+1}$，即 $-S_n=2+2^2+2^3+\cdots+2^n-n\cdot 2^{n+1}$。
4. 括号内是首项 2、公比 2 的等比数列，和为 $\dfrac{2(2^n-1)}{2-1}=2^{n+1}-2$。
5. 故 $-S_n=(2^{n+1}-2)-n\cdot 2^{n+1}$，$S_n=n\cdot 2^{n+1}-2^{n+1}+2=(n-1)2^{n+1}+2$。

---

### 4.5 分组求和法

**适用条件**：数列 $\{c_n\}$ 可写为 $c_n=a_n+b_n$，其中 $\{a_n\}$、$\{b_n\}$ 分别为等差、等比或其他可求和数列。

**操作**：
$$S_n=\sum_{k=1}^n c_k=\sum_{k=1}^n a_k+\sum_{k=1}^n b_k=S_n^{(a)}+S_n^{(b)}.$$

**示例**：$c_n=(2n-1)+3^n$。$\sum (2n-1)=n^2$，$\sum 3^n=\dfrac{3(3^n-1)}{2}$，故 $S_n=n^2+\dfrac{3(3^n-1)}{2}$。

---

### 4.6 并项求和法

**适用条件**：摆动数列（符号交替）或周期性数列。

**操作**：相邻两项并项求和，把符号问题转化为通项结构问题。

**示例**：$S_n=1-2+3-4+\cdots+(-1)^{n+1}n$。
- $n$ 为偶数：$S_n=(1-2)+(3-4)+\cdots+((n-1)-n)=(-1)\cdot\dfrac n2=-\dfrac n2$。
- $n$ 为奇数：$S_n=(1-2)+(3-4)+\cdots+((n-2)-(n-1))+n=-\dfrac{n-1}{2}+n=\dfrac{n+1}{2}$。

---

### 4.7 倒序相加法的推广

**适用**：首末两端项之和为定值的数列。

**示例**：求 $S_n=C_n^0+C_n^1+\cdots+C_n^n$。
1. 由组合数对称性 $C_n^k=C_n^{n-k}$（依据组合数性质）。
2. $S_n=C_n^0+C_n^1+\cdots+C_n^n$。
3. 倒序 $S_n=C_n^n+C_n^{n-1}+\cdots+C_n^0$。
4. 两式相加 $2S_n=2(C_n^0+\cdots+C_n^n)=2S_n$，此法不直接给出值，但结合二项式定理 $S_n=(1+1)^n=2^n$。
5. 关键是利用 $C_n^k=C_n^{n-k}$ 的对称性，是倒序相加思想在组合数中的体现。

---

## 五、等差数列前 n 项和的最大值

### 5.1 判定方法

**情形**：等差数列 $\{a_n\}$，公差 $d\ne 0$，$S_n=\dfrac{d}{2}n^2+\big(a_1-\dfrac{d}{2}\big)n$。

**方法一：二次函数对称轴法**
1. $S_n$ 是关于 $n$ 的二次函数 $S_n=\dfrac{d}{2}n^2+\big(a_1-\dfrac{d}{2}\big)n$，常数项为 0。
2. 对称轴 $n=-\dfrac{b}{2a}=-\dfrac{a_1-\frac{d}{2}}{2\cdot\frac{d}{2}}=\dfrac{\frac{d}{2}-a_1}{d}=\dfrac12-\dfrac{a_1}{d}$。
3. 若 $d<0$（开口向下），$S_n$ 有最大值，取最接近对称轴的正整数 $n$ 使 $S_n$ 最大。
4. 若 $d>0$（开口向上），$S_n$ 有最小值。

**方法二：单项符号法（更常用）**
1. 因 $S_n-S_{n-1}=a_n$，$S_n$ 递增 $\Leftrightarrow$ $a_n>0$；$S_n$ 递减 $\Leftrightarrow$ $a_n<0$。
2. 设 $d<0$（$\{a_n\}$ 递减）：
   - 当 $a_n\ge 0$ 且 $a_{n+1}<0$ 时，$S_n$ 达到最大值。
   - 即找到使 $a_n\ge 0$ 的最大 $n$：由 $a_n=a_1+(n-1)d\ge 0$ 得 $n\le 1-\dfrac{a_1}{d}$（注意 $d<0$，不等号方向）。
3. 设 $d>0$（$\{a_n\}$ 递增）：
   - 当 $a_n\le 0$ 且 $a_{n+1}>0$ 时，$S_n$ 达到最小值。

**推导**（方法二依据）：
1. $S_{n+1}-S_n=a_{n+1}$（依据 1.1）。
2. 若 $a_{n+1}<0$，则 $S_{n+1}<S_n$，即 $S$ 在 $n$ 之后开始减少；若 $a_n\ge 0$，则 $S_n\ge S_{n-1}$，即 $S$ 在 $n$ 之前还在增加。
3. 故 $a_n\ge 0$、$a_{n+1}<0$ 同时成立时，$S_n$ 是"由增转减"的转折点，即最大值。

**特殊情况**：若存在 $n_0$ 使 $a_{n_0}=0$（即 $a_1+(n_0-1)d=0$），则 $S_{n_0-1}=S_{n_0}$ 同时为最大值（因 $a_{n_0}=0$ ⇒ $S_{n_0}=S_{n_0-1}$，而 $a_{n_0+1}<0$）。

### 5.2 示例

**示例**：等差数列 $a_1=20$，$d=-3$，求 $S_n$ 的最大值。
1. $a_n=20-3(n-1)=23-3n$。
2. 令 $a_n\ge 0$：$23-3n\ge 0$，$n\le\dfrac{23}{3}\approx 7.67$，故 $n=7$ 时 $a_7=23-21=2>0$。
3. $a_8=23-24=-1<0$。
4. 故 $S_7$ 最大，$S_7=7\cdot 20+\dfrac{7\cdot 6}{2}(-3)=140-63=77$。

**对称轴法验证**：$S_n=-\dfrac32 n^2+\big(20+\dfrac32\big)n=-\dfrac32 n^2+\dfrac{43}{2}n$，对称轴 $n=\dfrac{43/2}{2\cdot 3/2}=\dfrac{43}{6}\approx 7.17$，最接近的整数为 $n=7$，与上法一致。

---

### 5.3 等差数列前 n 项积的最大值（补充）

**情形**：当 $\{a_n\}$ 各项为正且递减（$d<0$，$a_1>0$），求 $\prod\limits_{k=1}^n a_k$ 的最大值。

**思路**：积 $P_n=a_1 a_2\cdots a_n$。$P_{n+1}=P_n\cdot a_{n+1}$。
1. 当 $a_{n+1}>1$ 时 $P_{n+1}>P_n$，积增大。
2. 当 $0<a_{n+1}<1$ 时 $P_{n+1}<P_n$，积减小。
3. 当 $a_{n+1}=1$ 时 $P_{n+1}=P_n$，并列最大。
4. 故最大值出现在 $a_n\ge 1$ 且 $a_{n+1}\le 1$ 时（若 $a_{n+1}=1$ 则 $P_n=P_{n+1}$ 并列最大）。

**推导依据**：$\dfrac{P_{n+1}}{P_n}=a_{n+1}$（依据 $P_{n+1}=P_n\cdot a_{n+1}$），比较 $a_{n+1}$ 与 1 的大小决定积的增减。

## 小结与高频考点提示

1. **$a_n$ 与 $S_n$ 关系**：$a_n=S_n-S_{n-1}$（$n\ge 2$），**必须验证 $n=1$**，这是高考小题最高频陷阱。当 $S_n$ 含常数项（如 $S_n=2^n+1$）时，$n=1$ 往往不吻合，须写分段。
2. **等差五大公式**：通项（累加法）、前 n 项和（倒序相加）、性质 $a_m+a_n=a_p+a_q$（$m+n=p+q$）必须会推。$S_n$ 是 $n$ 的二次函数（常数项 0）是判定最值的根。
3. **等比五大公式**：通项（累乘法）、前 n 项和（错位相减，**必讨论 $q=1$ 与 $q\ne 1$**）、性质 $a_m a_n=a_p a_q$（$m+n=p+q$）。$S_n,S_{2n}-S_n,S_{3n}-S_{2n},\ldots$ 成等比是高频小题性质。
4. **裂项相消**：$\dfrac{1}{n(n+k)}=\dfrac{1}{k}\!\left(\dfrac1n-\dfrac{1}{n+k}\right)$ 用待定系数法推导，求和时注意首尾保留项。常见还有根式裂项 $\dfrac{1}{\sqrt n+\sqrt{n+1}}=\sqrt{n+1}-\sqrt n$、对数裂项 $\lg\dfrac{n+1}{n}=\lg(n+1)-\lg n$、阶乘裂项 $\dfrac{n}{(n+1)!}=\dfrac1{n!}-\dfrac1{(n+1)!}$。
5. **错位相减**：仅适用于"等差 × 等比"型 $b_n=(A+Bn)q^n$；步骤：写出 $S_n$ → 乘 $q$ → 相减 → 用等比求和 → 除以 $1-q$。注意首项与末项保留、中间项系数为 $d$。
6. **分组求和、并项求和**：分组适用于 $c_n=a_n+b_n$ 可拆分；并项适用于符号交替或周期性数列。
7. **$S_n$ 最值**：$d<0$ 时用 $a_n\ge 0$ 且 $a_{n+1}<0$ 定最大值位置；或用二次函数对称轴取最近整数点。若 $a_{n_0}=0$ 则 $S_{n_0-1}=S_{n_0}$ 同时最大。
8. **数列与函数综合**：$a_n$ 是 $n$ 的函数（等差是一次、等比是指数），$S_n$ 是 $n$ 的二次（等差）或指数型（等比）函数，可用函数方法研究最值与单调性。
9. **通项求法总结**：①已知 $S_n$ 用 $a_n=S_n-S_{n-1}$（验 $n=1$）；②累加法（$a_{n+1}-a_n=f(n)$ 型）；③累乘法（$\dfrac{a_{n+1}}{a_n}=f(n)$ 型）；④构造新数列（$a_{n+1}=pa_n+q$ 构造等比 $a_{n+1}-\alpha=p(a_n-\alpha)$，$\alpha=\dfrac{q}{1-p}$）；⑤待定系数法（设 $a_n=An^2+Bn+C$ 等多项式型）。
10. **求和方法总结**：①公式法（等差、等比直接套）；②裂项相消（分式、根式、对数、阶乘）；③错位相减（等差×等比）；④分组求和（可拆分）；⑤并项求和（符号交替）；⑥倒序相加（首末配对为定值）。

11. **$a_{n+1}=pa_n+q$ 型递推构造法**：设 $a_{n+1}-\alpha=p(a_n-\alpha)$，解 $\alpha=p\alpha+q$ 得 $\alpha=\dfrac{q}{1-p}$（$p\ne 1$）。则 $\{a_n-\alpha\}$ 是以 $a_1-\alpha$ 为首项、$p$ 为公比的等比数列，$a_n-\alpha=(a_1-\alpha)p^{n-1}$，故 $a_n=\alpha+(a_1-\alpha)p^{n-1}$。这是高考解答题的高频套路，务必熟练。

12. **数列综合题审题顺序**：先判断数列类型（等差/等比/其它）→ 选择通项求法 → 选择求和方法 → 注意 $n=1$ 验证与 $q=1$ 讨论 → 涉及最值时回归函数视角（$S_n$ 的二次函数性、$a_n$ 的符号变化）。

13. **易错点警示**：等比数列求和忘讨论 $q=1$；由 $S_n$ 求 $a_n$ 忘验证 $n=1$；裂项相消忘保留首尾项；错位相减时首末项符号弄反；$a_n$ 符号法判定最值时忽略 $a_{n_0}=0$ 的并列情形。
