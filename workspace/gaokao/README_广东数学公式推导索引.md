## 使用建议

- **查漏补缺**：按序号通读，重点看自己薄弱模块的推导过程。
- **冲刺复习**：每个文件末尾"小结与高频考点提示"独立可读，优先看。
- **压轴攻坚**：03 导数 + 09 解析几何两个文件为压轴双主线，公式与推导最详尽。
- **跨模块**：13 情境应用文件串联其他模块，适合做综合题前回顾。
